# word-count-clj

## Installation

* Install Java 8 or 9
* Install [Leiningen](https://leiningen.org/)

## Build

```sh
lein compile
lein uberjar
```

## Run (with java)

```sh
java -jar target/uberjar/word-count-clj-0.1.0-SNAPSHOT-standalone.jar
```

## Run (with lein)

```sh
lein run -m word-count-clj.core
```
